# UnrealTalks_OpenSource
demo project used for the talks


```git clone https://github.com/UnrealTalks/UnrealTalks_OpenSource.git```

after it clones the project run:
```git submodule init```
followed by:
```git submodule update```
to clone the plugin submodule
